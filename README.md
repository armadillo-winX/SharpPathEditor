# SharpPathEditor
## これは何?
Windows の環境変数の PATH を手軽に編集するためのツールです。<br>
コマンドラインツールなどを PATH に通すときに設定画面にアクセスするのがちょっと手間だな…と感じたので作ってみました。
## 推奨動作環境
- Microsoft Windows 10
- Microsoft Windows 11<br>
.NET Desktop Runtime 6.0 が必要です。
